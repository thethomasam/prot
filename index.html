    SCellBOW: Enhancing Single-Cell RNA-seq Analysis with Language Model Approaches

    SCellBOW (short for Single Cell Bag of Words) is an unsupervised transfer learning algorithm for clustering scRNA-seq data and phenotype algebra analysis on the clusters using distributed Bag-of-Words. SCellBOW enables the transformation of scRNA-seq data expression matrices to a low-dimensional vector space, where genes are analogous to words, and cells are analogous to documents. Construction of these cell documents enables the direct application of tools from Natural Language Processing (NLP) to the analysis of single-cell sequencing data. SCellBOW shows that NLP in amalgation with transfer learning allows a better representation of cell clusters which can help to decipher the heterogeneity of cells, especially in cancer genomics. SCellBOW facilitates robust identification of malignant clusters and enables stratification of the clusters based on their survival-risk.

    Adaptive Weighted Supervised Autoencoder for CNC Cutting Machine Simulation (Digital Twin)

    This project which was part of the Pawsey Research Fellowship, showcases the development of an adaptive weighted supervised autoencoder. The core objective of this autoencoder is to masterfully learn a low-dimensional representation of input data, simultaneously predicting a range of target variables. What sets this model apart is the dynamic update of the loss function's weighting during training. This critical feature ensures a balanced focus on both reconstruction loss and regression loss, optimizing the model's performance. We've adeptly applied this technology to simulate the operations of a CNC meat cutting machine, demonstrating its practical application in a real-world scenario. The results exhibit the autoencoder's capability to not only understand complex data patterns but also to accurately predict outcomes, making it a valuable tool in the field of machine learning and automation.

    Precision Agriculture Enhancement with SegGPT and YOLOv8

    This project introduces an advanced plant counting algorithm designed for UAS data, utilizing the latest in object detection through YOLOv8 and UAS photogrammetry. Our approach uniquely identifies and geolocates various plant sizes and conditions, directly from UAS images, generating a detailed spatial map for cluster analysis. A standout feature is the use of SegGPT for precise stubble detection and the integration of NVI for effective weed identification. These techniques collectively optimize crop layout understanding, aiding in the improvement of agricultural strategies such as irrigation efficiency and nutrient distribution. This project demonstrates significant advancements in precision agriculture, offering insightful contributions to both crop management and agricultural practice optimization."
